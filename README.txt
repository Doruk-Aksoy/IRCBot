ABOUT

    This is a simple IRCBot that uses an existing framework (PIRCBot) to handle the communication in the back with an event driven system.
    It allows easy extension of features, command recognition and chat based game handling.

REQUIREMENTS

* Requires pircbot-1.5.0 and JDBC (ojdbc6) libraries to compile.
    - PIRCBot: http://www.jibble.org/pircbot.php
    - JDBC: http://www.oracle.com/technetwork/apps-tech/jdbc-112010-090769.html
